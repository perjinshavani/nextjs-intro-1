# Nature Explorer

## Om projektet

Nature Explorer är en webbapplikation byggd med Next.js, React och TypeScript. Projektet presenterar olika platser i Sverige med hjälp av återanvändbara komponenter och responsiv design.

## Funktioner

- Visar olika platser i Sverige
- Återanvändbara React-komponenter
- Routing med Next.js
- Responsiv design

## Tekniker

- Next.js
- React
- TypeScript
- Tailwind CSS
- Git & GitHub

## Det här har jag lärt mig

Genom projektet har jag lärt mig att arbeta med komponenter, routing, layouts och responsiv design i Next.js.
